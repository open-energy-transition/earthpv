Pestimated installed rooftop capacity kWp → MWp

Adet counts only pixels above the 0.30 threshold that fall on a building footprint — the precision-honest floor. Aexp integrates sub-threshold probability, so it leans high. Reported capacity is P = A × η for each, giving the detected / expected pair (4,720 / 4,163 MWp nationwide) that brackets the true installed capacity.
